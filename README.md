# Using it (after install yabai and skhd)
```
git clone git@github.com:rsnullptr/yabai-skhd-rc.git ~/.config/yabai-skhd-rc ;\
cd ~/.config/yabai-skhd-rc ;\
ln -s ~/.yabairc yabairc ;\
ln -s ~/.skhdrc skhdrc ;\
```
